A dogebased style language
===========

## proposal

### many token
```css
many {
  background: red
}

* {
  background: red
}
```

### very token
```css
very doge{
  background: red
}

.doge {
  background: red
}
```

### such token
```css
such doge{
  background: red
}

#doge {
  background: red
}
```
